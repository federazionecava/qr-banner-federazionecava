# Linktree · Estilo Veneto

Página de enlaces simple, hecha en HTML y CSS puro, con una paleta inspirada en la región del Véneto.

## Estructura

```
.
├── index.html   # Markup de la página
└── style.css    # Estilos (colores, tipografía, layout)
```

## Uso

1. Abrí `index.html` en el navegador, o subilo a cualquier hosting estático (GitHub Pages, Netlify, Vercel, etc.).
2. Editá los `href` de cada enlace en `index.html` con tus redes reales (Instagram, Facebook, sitio web, mail, WhatsApp).
3. Cambiá el nombre y subtítulo en el `<h1>` y `.subtitle`.
4. Opcional: reemplazá el `.avatar` (círculo con gradiente) por una imagen real con `<img src="tu-foto.jpg">`.

## Personalización

Los colores y tipografías están centralizados como variables CSS al inicio de `style.css` (`:root`), así que se pueden ajustar fácil sin tocar el resto del código.

## Tech

- HTML5
- CSS3 (sin frameworks)
- Google Fonts (Cormorant Garamond + Jost)
